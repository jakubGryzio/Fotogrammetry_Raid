How to use the application

The photogrammetry raid planning is used to select the proper camera and plane and set the right parameter to visualize photogrammetry raid

At the settings page, you can choose the camera and the plane, then go to the parameter page by clicking the button.

You can set proper parameters to visualize raid at the parameters page.

In order to see visualize the photogrammetry raid click the button, then you will be moved to visualization page.

If you would like to change settings, choose the right page in the menu.
